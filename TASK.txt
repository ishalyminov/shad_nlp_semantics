DONE 2) what consists of <ingredient> (response: a list of dishes)

DONE 3) does <dish> consists of <ingredient - one or many>? (response: YES/NO)

DONE 4) is <dish> vegetarian (response - YES/NO)

DONE 5) is <dish> kosher (response - YES/NO)

DONE PART 6) how many calories does <dish/ingredient> have (response: a number, for a dish: the sum of nutrition for all the ingredients)

DONE 7) how many dishes are vegetarian (response: a number)

DONE 8) how many dishes are kosher (response: a number)

DONE 9) how many dishes consist of <ingredient>? (response: a number)

DONE 10) how many ingredients does <dish> consists of? (response: a number)

DONE 11) how long does <dish> take (response: cooking time, a string or number)

12) how much is <dish/ingredient> (response: a number, for a dish: the sum of costs for all the ingredients)
